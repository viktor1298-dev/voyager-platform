**Create `apps/api/src/routes/pod-terminal.ts`** (per D-02, RESEARCH Pattern 1):

Import `Exec` from `@kubernetes/client-node`, `PassThrough` from `node:stream`, `auth` from lib, `clusterClientPool` from lib.

Export `async function registerPodTerminalRoute(app: FastifyInstance): Promise<void>`

Route: `app.get('/api/pod-terminal', { websocket: true }, async (socket, request) => { ... })`

Implementation:
1. **Authenticate** — Extract session from request headers using `auth.api.getSession({ headers: request.headers })`. If no session, `socket.close(1008, 'Unauthorized')` and return.
2. **Parse query params** — `clusterId`, `namespace`, `podName`, `container` from `request.query`. Validate with Zod.
3. **Get KubeConfig** — `const kc = await clusterClientPool.getClient(clusterId)`
4. **Create PassThrough streams** — `stdout`, `stderr`, `stdin`
5. **Forward stdout/stderr to WebSocket** — `stdout.on('data', (data: Buffer) => { if (socket.readyState === 1) socket.send(data) })`. Same for stderr.
6. **Forward WebSocket messages to stdin** — `socket.on('message', (data) => stdin.write(data))`
7. **Attempt shells in order** — Try `/bin/bash`, `/bin/sh`, `/bin/ash`. For each shell, create `new Exec(kc)` and call `exec.exec(namespace, podName, container, [shell], stdout, stderr, stdin, true, (status) => { socket.close(1000, JSON.stringify(status)) })`. On success break, on error continue.
8. If all shells fail: `socket.close(1008, 'No shell available in container')`
9. **Cleanup on disconnect** — `socket.on('close', () => { stdin.end(); k8sWs?.close() })`

Register in server.ts: `import { registerPodTerminalRoute } from './routes/pod-terminal.js'` and call `await registerPodTerminalRoute(app)` after WebSocket plugin registration.

Run `pnpm typecheck` on api to verify.
