    **WatchManager class methods:**
    - `subscribe(clusterId: string): Promise<void>` — Increment subscriberCount. If first subscriber (count goes from 0 to 1), create informers for all 15 types using `clusterClientPool.getClient()` and `k8s.makeInformer()`. On each informer: register `add`, `update`, `delete` callbacks that call the appropriate mapper and emit via `voyagerEmitter.emitWatchEvent()`. Register `error` callback with exponential backoff reconnect. Register `connect` callback to reset reconnect attempts. Call `informer.start()` for each. Respect `MAX_CONCURRENT_CLUSTER_WATCHES` — if limit reached, log warning and skip.
    - `unsubscribe(clusterId: string): void` — Decrement subscriberCount. If reaches 0, call `informer.stop()` for all types and remove from map.
    - `getResources(clusterId: string, type: ResourceType): ReadonlyArray<k8s.KubernetesObject>` — Return `informer.list()` from the cluster's informer for that type. Return empty array if cluster not watched or informer not found.
    - `getResource(clusterId: string, type: ResourceType, name: string, namespace?: string): k8s.KubernetesObject | undefined` — Return `informer.get(name, namespace)`.
    - `isWatching(clusterId: string): boolean` — Return true if subscriberCount > 0.
    - `getActiveClusterIds(): string[]` — Return cluster IDs with active watches.
    - `stopAll(): void` — Stop all informers for all clusters. Used on server shutdown.

    **Reconnection logic (in error handler):**
    ```typescript
    private handleInformerError(clusterId: string, type: ResourceType, err: unknown): void {
      const cluster = this.clusters.get(clusterId)
      if (!cluster) return
      const attempt = (cluster.reconnectAttempts.get(type) ?? 0) + 1
      cluster.reconnectAttempts.set(type, attempt)
      const delay = Math.min(WATCH_RECONNECT_BASE_MS * 2 ** (attempt - 1), WATCH_RECONNECT_MAX_MS)
      const jitter = delay * WATCH_RECONNECT_JITTER_RATIO * Math.random()
      console.warn(`[WatchManager] Informer error for ${type} on cluster ${clusterId}, reconnecting in ${Math.round(delay + jitter)}ms (attempt ${attempt})`)
      setTimeout(() => {
        if (this.isWatching(clusterId)) {
          const informer = cluster.informers.get(type)
          informer?.start().catch((startErr) => this.handleInformerError(clusterId, type, startErr))
        }
      }, delay + jitter)
    }
    ```
